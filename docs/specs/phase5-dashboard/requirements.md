# Zeus Dashboard - 要件定義書

## 概要

Zeus CLI の Web ダッシュボード機能。ブラウザでプロジェクト状態を可視化する。

## 機能要件

### コマンドインターフェース

```bash
zeus dashboard [--port PORT] [--no-open]
```

| オプション | 型 | デフォルト | 説明 |
|-----------|-----|----------|------|
| --port | int | 8080 | HTTP サーバーのポート番号 |
| --no-open | bool | false | ブラウザを自動で開かない |

### ダッシュボード機能

#### プロジェクト概要
- プロジェクト名、説明、開始日
- 全体進捗率（プログレスバー）
- 健全性ステータス（Good/Fair/Poor）

#### タスク統計
- 全タスク数、完了、進行中、保留
- カード形式で視覚化

#### タスク一覧
- テーブル形式（ID, タイトル, ステータス, 優先度, 担当者）
- ステータスによる色分け

#### 依存関係グラフ
- Mermaid.js によるインタラクティブグラフ
- ステータスによるノード色分け
- 循環依存の警告表示

#### 予測分析
- 完了予測日、信頼度、誤差範囲
- リスクレベルと要因一覧
- ベロシティトレンド

### REST API

| Method | Path | 説明 |
|--------|------|------|
| GET | / | ダッシュボード HTML |
| GET | /api/status | プロジェクト状態 |
| GET | /api/tasks | タスク一覧 |
| GET | /api/graph | 依存関係グラフ |
| GET | /api/predict | 予測分析結果 |

### リアルタイム更新
- 5秒間隔で自動更新（Polling）
- 接続状態インジケータ
- エラー時のリトライ機構

## 非機能要件

### パフォーマンス
- ダッシュボード初期表示: < 500ms
- API レスポンス: < 200ms

### 互換性
- ブラウザ: Chrome, Firefox, Safari, Edge（最新版）
- OS: macOS, Linux, Windows

### セキュリティ
- ローカルホスト専用（127.0.0.1 バインド）
- 読み取り専用 API

### 可用性
- グレースフルシャットダウン
- SIGINT/SIGTERM ハンドリング

## 技術制約

- Go 標準ライブラリのみ（net/http, embed）
- 外部フレームワーク不使用
- Mermaid.js は CDN から読み込み
- 既存の DI/Context パターンに準拠
