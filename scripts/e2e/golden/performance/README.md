# Performance Baselines (Future)

このディレクトリは将来のパフォーマンス検証用です。

## 現在の状況

現在、`window.__VIEWER_METRICS__` からメトリクスを収集していますが、
検証は行っていません（情報収集のみ）。

## 将来的な実装予定

- メトリクスのベースライン管理
- 統計的な異常検出
- パフォーマンス回帰レポート

## 収集しているメトリクス

テスト実行時に以下のメトリクスを収集:

- 初期描画時間
- ノード/エッジ描画時間
- インタラクション応答時間
- メモリ使用量

これらは `KEEP_ARTIFACTS=true` で実行した場合に
`/tmp/zeus-e2e-artifacts/metrics.json` に保存されます。
